# React
## Web Application
- 웹 애플리케이션의 규모가 커짐에 따라 UI, UX를 구현하기 위한 전통적 방법으로 한계가 생김. 
- 이를 극복하기 위한 다양한 Frontend Framwork 등장 → React, Vue, ...
- 기본적으로 JS, CSS, HTML 등이 필요하다.

### 구조
- 구조를 만들 때는 HTML 
    - 별로 이쁘지 않다
- 디자인을 꾸미는 것은 CSS
    - 웹페이지의 미적인 요소 구성
- 클라이언트-웹페이지 상호작용은 JS
    - 사용자가 클릭을 해 페이지가 넘어가고, ID PW를 입력해 로그인하는 등의 상호작용.

### 웹 애플리케이션 개발
- 프론트엔드 + 백엔드
- 프론트엔드
    - HTML
    - CSS
    - JS(javascript) > REACT의 기반. 
- 백엔드
    - DB. MySQL
    - JAVA ...
